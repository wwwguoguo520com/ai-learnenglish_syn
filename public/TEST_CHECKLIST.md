# 功能测试清单 ✅

## 快速开始测试

### 1. 打开文件
```
打开 index_v4.1_clean.html
```

### 2. 检查控制台（F12）
应该看到：
```
✅ 快速标注功能已加载 v4.1
✅ 快速标注功能已就绪
```

### 3. 测试快速标注

**步骤：**
1. 左侧粘贴文本：`Learning English is important.`
2. 点击"保存并打开短文"
3. 选中 "Learning"
4. 点击 "⭐ 生词"
5. 应该看到绿色提示 "✓ 生词标注已保存"
6. "Learning" 应该变为黄色加粗带星标

### 4. 测试快捷键

**步骤：**
1. 选中 "English"
2. 按 `Ctrl+1` (Windows) 或 `Cmd+1` (Mac)
3. 应该看到绿色提示并标注成功

### 5. 测试其他快速标注

- ⭐ 生词 → 黄色 + 加粗 + ⭐
- 💡 重点 → 蓝色 + 加粗 + 💡 + 方框
- ❗ 疑难 → 橙色 + 加粗 + ❗ + 圆框
- 📝 词组 → 绿色 + 下划线
